## Mój pierwszy projekt w Django

Miała być to strona Muzeum Telefonii Komórkowej w Sosnowcu

## Technologie
Projekt jest stworzony w oparciu o Django (MVT):
* Python 3.10
* Django 3.2.3
* Bootstrap 5
* TinyMCE
* Docker
* Docker-compose

## Baza danych
* PostgreSQL 14.2

## Instalacja

Aby pobrać projekt:
```
$ git clone https://github.com/Fejren/muzeum.git
```
Aby uruchomić projekt:
```
$ docker-compose up
```
lub
```
$ make up
```
